# Algorithm Visualizer

## 概要
ソートアルゴリズム（クイックソート・マージソート）を実装し、
実行時間の比較ができるアプリケーション。

## 作成目的
アルゴリズムの理解を深めるとともに、
ソフトウェア設計（責務分離）を意識した開発を行うため。

## 機能
- クイックソート
- マージソート
- 実行時間計測

## 技術スタック
- Python

## ディレクトリ構成
project/

│   │   ├── quick_sort.py   # クイックソート

## アルゴリズム比較

| アルゴリズム   | 計算量（平均） | 安定性   | 特徴 |
|--------------|--------------|--------|------|
| QuickSort    | O(n log n)   | ×      | 高速・in-place |
| MergeSort    | O(n log n)   | ○      | 安定・追加メモリ使用 |

## 工夫した点
- ソートアルゴリズムをクラスとして抽象化
- publicメソッドと内部処理を分離
- 元データを変更しない設計
- 再帰処理の責務を明確化
