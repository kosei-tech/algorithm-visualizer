# Algorithm Visualizer

## 概要
ソートアルゴリズム（クイックソート・マージソート）を実装し、
実行時間の比較ができるアプリケーション。

## 作成目的
アルゴリズムの理解を深めるとともに、
ソフトウェア設計（責務分離）を意識した開発を行うため。

## 機能
- クイックソート
- マージソート
- 実行時間計測

## 技術スタック
- Python

## ディレクトリ構成
project/
├── algorithms/              # アルゴリズム層（純粋ロジック）
│   └── sort/
│       ├── sorter.py       # 抽象クラス（インターフェース）
│       ├── quick_sort.py   # クイックソート
│       └── merge_sort.py   # マージソート
│
├── services/               # ビジネスロジック層
│   └── sort_service.py     # ソート処理の管理・時間計測
│
├── main.py                 # エントリーポイント（CLI）
│
└── README.md               # プロジェクト説明
